📊 Attendance Tracker

A role-based attendance management system with secure APIs and cloud deployment.

🔍 Problem

Many companies struggled to track employee attendance securely and needed real-time dashboards.

⚡ Solution

I built a Spring Boot backend with JWT authentication, integrated MongoDB for scalability, and a React dashboard for admins & employees.

🛠️ Tech Stack

React, Spring Boot, MongoDB, JWT, Docker

📈 Impact

Reduced manual tracking by 90%, improved accuracy, and enabled remote access for HR managers.

🎓 Training App
